# Browser Sound to MP3 (Chrome Extension)

ボタンを押してPC再生音を録音し、MP3として保存するChrome拡張です。

## 仕様メモ

- 録音開始時にChromeの画面共有ダイアログが開きます
- そこで**システム音声共有をON**にしてください
- 出力先がスピーカーでもイヤホンでも、共有されたシステム音声を録音します

## 使い方

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」でこのディレクトリを選ぶ
4. 拡張のポップアップを開いて `録音開始`
5. 画面共有ダイアログで画面と音声共有を許可
6. 録音終了後 `録音停止` を押す
7. MP3保存ダイアログが開く

## ローカルテスト

```bash
npm install
npm test
```

## 補足

- Chromeの制約で、ユーザー操作なしのバックグラウンド自動録音はしていません
- MP3エンコードには `lamejs` を同梱して使用しています
