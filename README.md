# Browser Sound to MP3 (Chrome Extension)

PCで再生している音の録音（MP3）と、画面+音声キャプチャ（WebM）ができるChrome拡張です。

## 重要な挙動

- 拡張ポップアップは操作外クリックで閉じるため、録音は**専用録音ウィンドウ**で実行します
- 専用録音ウィンドウは、他のタブやアプリを操作しても録音が継続します
- 画面共有ダイアログでは**システム音声共有をON**にしてください
- スピーカーでもイヤホンでも、共有されたシステム音声を録音します

## 使い方

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」でこのディレクトリを選ぶ
4. 拡張ポップアップで `録音ウィンドウを開く` を押す
5. 開いた録音ウィンドウでモードを選ぶ
   - `音声のみ (MP3)`  
   - `画面+音声 (WebM)`
6. `開始` を押す
7. 画面共有ダイアログで対象画面と音声共有を許可
8. 終了時に `停止` を押す
9. モードに応じて保存ダイアログが開く（MP3 or WebM）

## ローカルテスト

```bash
npm install
npm test
```

## 補足

- Chromeの制約で、ユーザー操作なしのバックグラウンド自動録音はしていません
- MP3エンコードには `lamejs` を同梱して使用しています
